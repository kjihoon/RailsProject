## 6. Faker data(0626)

###  Input faker data

```ruby
#In Gemfile

gem 'faker', :git => 'https://github.com/stympy/faker.git', :branch => 'master' ###facker data 만들어주는 gem

#gem 'pry-rails'
```

```ruby
# db/seeds.rb

require 'faker'
10.times do
Post.create(
  title: Faker::Pokemon.name,
  content: Faker::OnePiece
)
end
```

```cmd
# In Ubuntu
# 10개의 faker data 입력됨
$ rake db:seed
```
